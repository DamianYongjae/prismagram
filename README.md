- [x] create account
- [x] request secret
- [x] confirm secret (login)
- [x] like / unlike a photo

- [ ] ses user profile
- [ ] follow / unfollow
- [ ] see the full photo
- [ ] edit my profile
- [ ] upload a photos
- [ ] edit the photo (delete )
- [ ] see the feed
