# prismagram
Instagram Clone with Express + Prisma + React  and React Native

## user stories

- [x] create account
- [x] request secret
- [x] confirm secret (login)
- [x] like / unlike a photo 
- [x] comment on a photo
- [x] search by user
- [x] search by location
- [x] follow user
- [x] unfollow user
- [x] edit my profile
- [ ] see user profile
- [ ] see my profile
- [ ] see the full photo
- [ ] upload a photos
- [ ] edit the photo (delete )
- [ ] see the feed
